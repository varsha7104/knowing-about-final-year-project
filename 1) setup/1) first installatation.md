## Installations
Open cmd and follow below steps
1) See if your system has a node in it
   ```
   node -v
   ```
   Output
   ```` v22.12.0 ````
   Here make sure that version is greater than 18
2) Also check if your system has npx and npm
   ```
   npx -v
   npm -v
   ```
   Output:
   ````
11.4.1
11.4.1 ````
3) Install next js using command

```
npx create-next-app@15.3.2
```
Make sure you install the same version so that there wont be disturbance in future
